# Tarkk.ai - General notes

## Report structure

- Vendor information
  - Country etc.
  - Company
- Description and nature of the app
  - Open information from the web
  - Optionally you can input documents/information that the vendor has provided
- Auth: MFA/SSO
- Platforms
  - Windows
  - Web
  - Mac
  - Android
- Data storage
  - Location of data centers
  - Data stored
  - Data in transit
  - Data policies
  - GDPR
- What type of permissions are required
  - SUDO
  - Camera access?
  - Microphone access?
- CVE
- Versions and updates
  - When was the last update
  - How often are updates
- Is there AI features
- Confidence rating
- Security certificates
- Any big data leaks / incidents

## Report types

- Short report
- Medium report
- Long report
- Enterprise report
